Hilfe zum Gerätetreibermanager

Beschreibung

Der Gerätetreibermanager ist ein Werkzeug, welches dem Nutzer bei der Installation von Hardwaretreibern von Dritthersteller unterstützt. Momentan können Treiber von Nvidia, ATI und Broadcom sowie der PAE Kernel für 32-bit Systeme installiert werden. Der Gerätetreibermanager listet nur unterstützte Geräte auf. Andere Treiber als oben erwähnt werden nicht angezeigt.

Wichtig

Der Gerätetreibermanager kann nur die Treiber installieren, die in den Repositorys vorhanden sind. Sollten Sie Treiber installieren wollen, sorgen Sie dafür, dass Sie eine Internetverbindung haben.

Befehle

Wenn Sie die Treiberinstallation im Terminal durchführen wollen, können Sie die folgenden Befehle benutzen:

 

# ddm -i ati

Das wird die fglrx Treiber für Ihre ATI Grafikkarte installieren, ATI Grafikkarten der Versionen HD 5000 und später werden durch die fglrx Treiber unterstützt. Versionen zwischen 1000 und 5000 nutzen die Radeon Treiber. Benutzen Sie "ddm -p ati" um die Treiber von Ihrem System zu entfernen.

# ddm -i nvidia

Das wird die passenden Treiber für Ihre Nvidia Grafikkarte wählen. Sollten Sie eine Hybridgrafikkarte besitzen (gleichzeit Nvidia und Intel) werden die Bumblebee Treiber ausgewählt. Benutzen Sie "ddm -p nividia" um die Treiber von Ihrem System zu entfernen.

# ddm -i open

Sie können diesen Befehl benutzen, wenn Sie zu den offenen Nouveau Treiber zurückkehren wollen. Er wird sämtliche proprietären Treiber von Ihrem System entfernen.

# ddm -i broadcom

Dieser Befehl wird die passenden Treiber für Ihre Broadcom W-LAN-Adapter installieren. Benutzen Sie "ddm -p broadcom" um die Treiber von Ihrem System zu entfernen.

# ddm -i pae

Das wird den PAE Kernel auf einem Multi-Prozessor.System mit einem 32-bit Betriebssystem OS installieren. Benutzen Sie "ddm -p pae" um den PAE Kernel von Ihrem System zu entfernen. Beachten Sie, dass der Kernel nicht deinstalliert werden kann, wenn Sie Ihr Betriebssystem gleichzeitig mit diesem in diesen Kernel gestartet haben.

# ddm -i fixbumblebee

Das wird die Reparatur einer Bumblebee Installation versuchen.
